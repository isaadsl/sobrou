# Sobrou

Sistema web de planejamento financeiro pessoal desenvolvido para auxiliar usuários no controle de receitas, despesas e organização financeira. O projeto tem como objetivo oferecer uma forma simples e intuitiva de visualizar o orçamento, acompanhar gastos e facilitar o planejamento financeiro do dia a dia.

Este projeto foi desenvolvido como parte dos estudos em **Análise e Desenvolvimento de Sistemas**, aplicando conceitos de desenvolvimento web, autenticação de usuários, banco de dados em nuvem e integração com serviços externos.

---

## Funcionalidades

* Cadastro de usuários com e-mail e senha
* Recuperação de senha
* Verificação de e-mail
* Autenticação segura utilizando Supabase
* Controle de receitas
* Cadastro de despesas fixas e variáveis
* Visualização do saldo disponível
* Interface responsiva para desktop e dispositivos móveis

## Tecnologias utilizadas

* HTML
* CSS
* JavaScript
* Supabase
* GitHub Pages

## Objetivos do projeto

* Desenvolver uma aplicação web para organização financeira.
* Aplicar conceitos de autenticação e gerenciamento de usuários.
* Utilizar um banco de dados em nuvem para armazenamento das informações.
* Desenvolver uma interface simples, acessível e responsiva.
* Consolidar conhecimentos adquiridos durante a graduação em Análise e Desenvolvimento de Sistemas.
